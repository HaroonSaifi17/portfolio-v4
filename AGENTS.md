# Project Context

This is a personal portfolio website built with Bun, Astro, Svelte, Tailwind CSS, shadcn-svelte, and MDX. The goal is a bold, high-quality portfolio that highlights projects, experience, blog posts, and case studies.

## Tech Stack

- Astro for routing, layouts, and content collections
- Svelte for interactive components
- Tailwind CSS for styling
- shadcn-svelte for UI components
- MDX for blog posts and case studies
- Bun for package management and scripts

## Content Locations

- Blog posts: `src/content/blog`
- Case studies: `src/content/case-studies`
- Layouts: `src/layouts`
- Global styles: `src/styles/global.css`

## Motion and Rendering

- GSAP drives scroll-triggered animations
- View Transitions are enabled in the main layout

## Notes

- Keep typography bold and editorial
- Prefer Astro for page structure and Svelte for interactivity
- Future additions: GSAP refinements and rehype-pretty-code for MDX
