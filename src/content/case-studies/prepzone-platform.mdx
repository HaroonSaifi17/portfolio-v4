## Overview

PrepZone is an EdTech assessment platform featuring secure Google OAuth, a large question repository for adaptive mock tests, and real-time performance analytics.

## Stack

- Angular
- Node.js
- Express
- MongoDB

## Highlights

- Implemented secure Google OAuth sign-in for student accounts.
- Built analytics dashboards with real-time performance insights.
