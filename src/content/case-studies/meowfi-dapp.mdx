## Overview

MeowFi DApp is a full-stack DeFi application featuring Liquidity Pools, Bribe Vaults, and token swaps via Balancer V3 protocols, built with end-to-end type safety across the stack.

## Stack

- React
- Node.js
- tRPC
- Web3

## Highlights

- Integrated Balancer V3 protocols for swaps and liquidity features.
- Maintained end-to-end type safety across frontend and backend flows.
